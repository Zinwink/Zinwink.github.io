python中__slots__的使用

Python __slots__

Python中普通实例属性都存储在__dict__字典中， 这种设计灵活，允许动态添加属性。但是每个实例都要维护一个字典，存在内存消耗和查找消耗。
使用__slots__显式指定类的属性，并禁止之后实例动态添加属性， 可以减少内存消耗，并优化访问速度。

使用__slots__时，python会

  1. 在类级别创建一个固定的内存布局，类似C语言中的结构体
  2. 不在为实例创建__dict__和__weakref__属性(除非显示添加到__slots__中)
  3. 将属性直接存储在预分配的固定大小的数组中，而不是字典里

由上，使用__slots__可以使属性访问更快(不需哈希查找)， 内存占用少， 没有__dict__的开销(每个实例节省一个字典内存)

与@dataclass相结合使用

@dataclass是Python 3.7 及更高版本中引入的一个装饰器， 它通过自动为类生成特殊方法，如__init__, __repr__, __eq__等简化类的创建过程。 这些方法基于类中定义的字段自动生成，减少了编写重复代码的需要。
